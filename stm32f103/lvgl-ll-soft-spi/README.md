### Lvgl graphic library running at STM32F103C8T6 BluePill board

- make
- make flash
- software SPI
- ILI9341 320x240 display
- Rotary encoder
- It is necessary to hack STM32F103C8T6 flash size from 64kB to 128kB in STM32F103C8Tx_FLASH.ld file
